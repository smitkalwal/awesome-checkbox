# awesome-checkbox
Jquery plugin to replace all checkboxes with custom awesome looking checkboxes.

## Example

<input type="checkbox" data-color="red" data-label="Options1" class="Option" value="">

<script>

	$('.Option').awesome({  // calls the init method
	  color : 'default'
	});

</script>
